Conway's Game of Life rules:

If a live cell has less than 2 live (friendly) cells adjacent, it dies.

If a live cell has more than 3 live (friendly) cells adjacent, it dies.

If an empty cell has exactly 3 live (friendly) cells adjacent, it becomes a live cell.

(Multiplayer) If a live cell has more than 2 live (enemy) cells adjacent, it dies.
